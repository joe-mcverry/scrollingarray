# ScrollingArray

[![Maven Central](https://img.shields.io/maven-central/v/io.github.joe-mcverry/scrollingarray.svg?label=Maven%20Central)](https://central.sonatype.com/artifact/io.github.joe-mcverry/scrollingarray)
[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://www.apache.org/licenses/LICENSE-2.0)
[![Java](https://img.shields.io/badge/Java-21%2B-orange)](https://www.java.com/)

**ScrollingArray** is a lightweight Java utility library that lets you easily generate combinatorial "scrolling" 
sub-arrays from both regular Java arrays and ArrayLists.

It supports multiple counting patterns and works with both primitive/object arrays and generic lists.

## Maven Dependency

```xml
<dependency>
    <groupId>io.github.joe-mcverry</groupId>
    <artifactId>scrollingarray</artifactId>
    <version>1.0.0</version>
</dependency>
```

### Features
- Simple, clean API
- Support for arrays and `ArrayList`s
- Generic type support (`<T>`)
- Multiple combinatorial patterns (Single Count, Multiple Counts, etc.)
- Fully unit tested (17 tests)
- Zero external dependencies
- Java 21+ compatible
- Apache License 2.0
